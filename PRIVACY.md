# Privacy Policy — PBS with BBA Compare

**Last updated:** March 8, 2026

## Overview

PBS with BBA Compare is a browser extension that adds Practice Bidding Scenarios (PBS) to Bridge Base Online (BBO). This policy describes what data the extension collects and how it is used.

## Data Stored Locally

The extension stores the following in your browser's localStorage. This data never leaves your device:

- **PBSCache** — cached scenario data for offline use
- **PBSSettings** — your settings preferences (toggle states)
- **Plugin configurations** — settings for PBS, BBA Compare, and PBN Capture plugins

## Data Collected by the BBA Compare Plugin

The BBA Compare plugin, loaded at runtime from a remote server, collects:

- **Anonymized IP address** — used for aggregate usage statistics
- **Scenario usage** — which practice scenarios are played
- **Auction comparison data** — bidding sequences submitted for comparison with the BBA bidding tool

This data is collected to improve the BBA Compare service and is not sold or shared with third parties.

## Data We Do Not Collect

- No personal information (name, email, account credentials)
- No browsing history
- No data from sites other than bridgebase.com

## Third-Party Code

The extension loads plugin scripts (BBA Compare, PBN Capture) from GitHub repositories at runtime. These plugins are maintained by known developers in the bridge community. The extension does not load any advertising or tracking scripts.
